# Gym App — Support & User Guide

## Quick Start Guide

### Intervals Tab

Create custom interval timers for your workouts.

1. Tap **+ New Interval** to create a timer.
2. Add steps — set each as **Work** or **Rest**, choose a duration, and pick a start sound.
3. Enable **3s countdown** to hear a countdown in the last 3 seconds of a step.
4. Enable **10s countdown** to hear the remaining time announced every 10 seconds (e.g. "Fifty", "Forty", etc.).
5. Tap **Start** on any interval to begin. The timer will guide you through each step with audio and haptic cues.

### Sessions Tab

Plan structured workouts with exercises, sets, and intervals.

1. Tap **+ New Session** to create a workout template.
2. Add exercises from the built-in library (~70 exercises across 7 categories).
3. Configure sets — choose from Standard, Pyramid, Drop Set, or AMRAP.
4. Link an interval timer to any exercise for timed work/rest periods.
5. Tap **Start** on a session to begin logging your workout. Track reps and weight as you go.
6. Tap **Finish Workout** when done.

### Settings Tab

- **Appearance** — Choose between System, Dark, or Light theme.
- **Accessibility** — Enable colour-blind friendly mode (orange/blue), high contrast, or reduce motion.
- **Voice** — Select which voice is used for audio countdown cues.
- **Exercises** — Filter the exercise library to show only the exercises you use.

---

## Tips

- **Muted mode**: When starting an interval from a session, tap the mute button (speaker icon) to run the timer silently — haptic feedback still works.
- **Supersets**: In the session editor, link two exercises together to pair them as a superset.
- All your data is stored locally on your device. Deleting the app will remove all saved intervals, sessions, and workout logs.
